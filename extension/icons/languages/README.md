# Language icons

Country flags are from `flag-icons` at commit
`086f7e97d657358203916dbe84f61c2bccaa81eb` and are distributed under the MIT
license in `flags/LICENSE`.

Language-to-flag mappings:

- Arabic: Saudi Arabia (`sa.svg`)
- Chinese: China (`cn.svg`)
- Czech: Czechia (`cz.svg`)
- Dutch: Netherlands (`nl.svg`)
- Spanish: Spain (`es.svg`)
- Greek: Greece (`gr.svg`)
- French: France (`fr.svg`)
- German: Germany (`de.svg`)
- Hindi: India (`in.svg`)
- Hungarian: Hungary (`hu.svg`)
- Indonesian: Indonesia (`id.svg`)
- Italian: Italy (`it.svg`)
- Japanese: Japan (`jp.svg`)
- Korean: South Korea (`kr.svg`)
- Polish: Poland (`pl.svg`)
- Portuguese: Portugal (`pt.svg`)
- Romanian: Romania (`ro.svg`)
- Russian: Russia (`ru.svg`)
- Turkish: Turkey (`tr.svg`)
- Ukrainian: Ukraine (`ua.svg`)
- Vietnamese: Vietnam (`vn.svg`)

Latin continues to use Vatican City (`va.svg`) only for existing profiles that contain saved vocabulary.
