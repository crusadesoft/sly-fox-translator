Sly Fox Translator Privacy Policy

Effective date: July 10, 2026

What Sly Fox Translator does

Sly Fox Translator is a Chrome extension for language learning. It replaces only vocabulary that you choose with contextually appropriate translations on webpages.

Information handled by the extension

  • Website content. The extension reads visible page text to find and replace the vocabulary you have selected. It asks Chrome's built-in Translator API to translate relevant page text. Sly Fox Translator does not operate a server that receives this text.
  • Your vocabulary and settings. Language profiles, vocabulary entries, definitions, excluded sites and pages, and extension settings are stored locally in Chrome using chrome.storage.local. They are not synced by this extension to a developer-operated server.
  • Optional word suggestions. When you explicitly use the Suggest feature, the English lookup term you enter and the selected language code are sent by HTTPS to Wikidata to retrieve candidate translations and definitions. The feature does not send page content or your full vocabulary list to Wikidata.

How information is used

Information is used only to provide the extension's language-learning features: translating relevant page content, applying your vocabulary list, preserving your settings, and showing optional word suggestions. Sly Fox Translator does not sell personal information, use it for advertising, or use it for creditworthiness or lending purposes.

Sharing

Sly Fox Translator does not sell or transfer your data to third parties. The optional Suggest feature makes a direct request to Wikidata only when you choose to use it. Chrome's built-in Translator API is provided by the browser; its operation is subject to Chrome's applicable terms and policies.

Retention and control

You can edit or delete vocabulary and excluded sites in the extension at any time. Removing the extension removes its locally stored extension data from Chrome.

Chrome Web Store policy

Sly Fox Translator's handling of user data complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.
